Kina

Edison desarrollos

Kina (conocido anteriormente como KineActiv) es un sistema que se enmarca en la tele-rehabilitación. Nació con el fin de promover el ejercicio físico y cognitivo de una manera intuitiva, sin cables ni sensores en el cuerpo del usuario. Está enfocado al público de la tercera edad con el ideal de retrasar la fragilidad motora. El sistema fue desarrollado con Unity 3D para crear un entorno virtual en el que el usuario interactua con elementos gamificados en realidad aumentada. Para el reconocimiento de las articulaciones se uso el periferico Kinect v2.0 y el SDK correspondiente de Microsoft.
